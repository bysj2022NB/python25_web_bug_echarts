## 介绍
1.多线程漏洞扫描，确保扫描高效性
2.扫描历史:可查看扫描历史任务并下载报告(pdf和word)
3.扫描配置:设置需要使用的漏洞类型，以及可以自定义添加漏洞，增加系统可扩展性
